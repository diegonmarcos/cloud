### Layer 1: Deep Architecture (Opus)

**When**: Building from zero, critical decisions

```yaml
triggers:
  - "design a system for..."
  - "architect the..."
  - "from scratch..."
  - "what's the best approach for..."
  - new project initialization
  - security-critical decisions

cost: $$$
frequency: 5-10% of tasks
value_to_student: HIGHEST (architectural wisdom)
```

**Training capture**:
- Full reasoning chains
- Alternative approaches considered
- Trade-off analysis
- Your feedback on decisions

### Layer 2: Middle Adjustment (Sonnet)

**When**: Maintenance, modifications, implementation

```yaml
triggers:
  - "modify this to..."
  - "refactor..."
  - "add feature..."
  - "fix this bug..."
  - "review this code..."
  - existing codebase work

cost: $$
frequency: 30-40% of tasks
value_to_student: HIGH (implementation patterns)
```

**Training capture**:
- Code transformations
- Refactoring patterns
- Bug fix approaches
- Your style preferences

### Layer 3: Easy Management (Haiku + Local)

**When**: Simple, routine, repetitive

```yaml
triggers:
  - "what's the command for..."
  - "quick script to..."
  - "format this..."
  - "convert..."
  - routine deployments
  - status checks

cost: $
frequency: 40-50% of tasks
value_to_student: MEDIUM (operational knowledge)
```

**Training capture**:
- Quick patterns
- Your shortcuts
- Common operations
- Frequently used commands

**Role**: Learn from everything, become you

## The Student's Curriculum

### Phase 1: Observation (Month 1-6)
- Collect all interactions
- Build preference profile
- Learn your vocabulary
- Map your repos

### Phase 2: Imitation (Month 6-12)
- Handle simple tasks (Layer 3)
- Shadow Sonnet (compare outputs)
- Learn from corrections
- Fine-tune on your style

### Phase 3: Independence (Year 1-2)
- Take over Layer 3
- Attempt Layer 2 (Sonnet verifies)
- Develop your "voice"
- Learn your architecture patterns

### Phase 4: Competence (Year 2-3)
- Handle Layer 2-3 alone
- Assist on Layer 1
- Proactive suggestions
- Pattern recognition

### Phase 5: Mastery (Year 3-5)
- Near-autonomous operation
- Opus only for novel problems
- Teaches new patterns to itself
- Your digital twin

### Phase 6: CEO (Year 5+)
- Runs your AI operations
- Makes decisions in your style
- Knows your entire history
- Occasionally consults Opus (board advisor)

class StudentTrainer:
    def monthly_finetune(self):

        data = self.load_training_data(last_30_days=True)

        # 2. Filter for quality
        gold = data.filter(quality="accepted", has_feedback=True)
        corrections = data.filter(has_user_edits=True)  # MOST VALUABLE

        # 3. Weight by layer
        weighted = WeightedDataset(
            architecture=gold.layer1 * 3.0,    # Rare, valuable
            implementation=gold.layer2 * 2.0,  # Common, important
            operations=gold.layer3 * 1.0,      # Frequent, routine
            corrections=corrections * 5.0      # YOUR STYLE = GOLD
        )

        # 4. LoRA fine-tune
        new_adapter = lora_finetune(
            base_model=self.current_model,
            dataset=weighted,
            epochs=3
        )

        # 5. Evaluate
        if self.evaluate(new_adapter) > self.current_score:
            self.deploy(new_adapter)

```python
class LayerRouter:
    def route(self, task: Task) -> Layer:
        # Layer 1: Deep Architecture
        if self.is_architectural(task):
            return Layer.OPUS

        # Layer 2: Middle Adjustment
        if self.is_implementation(task):
            # Try Student first if mature enough
            if self.student.competence_level >= 2:
                result = self.student.attempt(task)
                if result.confidence > 0.85:
                    return Layer.STUDENT
            return Layer.SONNET

        # Layer 3: Easy Management
        if self.student.competence_level >= 1:
            return Layer.STUDENT
        return Layer.HAIKU

    def is_architectural(self, task: Task) -> bool:
        signals = [
            "from scratch" in task.prompt,
            "design" in task.prompt,
            "architect" in task.prompt,
            task.is_new_project,
            task.is_security_critical,
            task.complexity_score > 0.8
        ]
        return sum(signals) >= 2
```
